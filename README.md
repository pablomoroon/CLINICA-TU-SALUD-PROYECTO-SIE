# CLINICA-TU-SALUD-PROYECTO-SIE

En este trabajo, realizado en la asignatura Sistemas de Información Empresariales mediante el uso de Bonita y Odoo, hemos implantado un sistema ERP y BPM en una clínica privada para la gestión integral de los procesos asistenciales y administrativos, incluyendo citas médicas, pruebas diagnósticas, facturación, compras e inventario, facilitando el trabajo del personal sanitario y administrativo y mejorando la atención al paciente de forma transparente.
